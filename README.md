# README GENERATOR

*user* will be prompted for their GitHub username and other several question about their project. and the README.md file  will be populated with the following :
* At least one badge
* Project title
* Description
* Table of Contents
* Installation
* Usage
* License
* Contributing
* Tests
* Questions
